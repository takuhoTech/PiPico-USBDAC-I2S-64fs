# PiPico-USBDAC-I2S-64fs
## What is this?
Pre-built usb_sound_card (pico-playground) modified to support 64fs YDA142 and other 64fs DACs.
## Wiring

| I2S       | GPIO  |
| --------- | ----- |
| I2S_BCK   | GP16  |
| I2S_LRCK  | GP17  |
| I2S_DATA  | GP18  |
